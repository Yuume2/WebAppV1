# V1 — MVP scope

## Included

- Auth (basic, single-user account)
- Connect API keys: OpenAI, Anthropic, Perplexity
- Projects (create, list, rename, delete)
- Workspace canvas with multiple chat windows per project
- Independent conversation context per window
- Conversation persistence

## Excluded (post-V1)

- Team / multi-user collaboration
- Billing / subscription layer
- Custom fine-tunes, RAG, file uploads
- Mobile app
- Plugin / extension ecosystem
- Shared provider keys / pooled billing
