# Vision — AI Workspace

## Problem

Users juggle multiple AI providers across disconnected tabs. No shared context, no structure, no persistence of their work.

## Solution

A single workspace where users:

- connect their own AI provider accounts via API (OpenAI, Anthropic, Perplexity)
- open several independent chat windows, each with its own model/context
- organize everything into projects on a workspace canvas

## Principles

- Bring-your-own-key — user owns data and cost
- Multi-provider by default — no lock-in
- Workspace-first UX — chats live inside projects, not in isolation
